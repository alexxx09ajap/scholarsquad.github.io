Estructura Básica de HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear páginas web. Se utiliza para estructurar el contenido en la web mediante etiquetas que indican cómo debe ser presentado.

Ejemplo de Estructura HTML


                <!DOCTYPE html>
                <html lang="es">
                    <head>
                        <meta charset="UTF-8">
                        <title>Ejemplo de Estructura HTML</title>
                    </head>
                    <body>
                        <h1>Hola, Mundo!</h1>
                        <p>Este es un ejemplo de un documento HTML básico.</p>
                        <a href="https://www.ejemplo.com">Visita Ejemplo</a>
                    </body>
                </html>

Etiquetas de Estructura Básica

  • <html>: Esta etiqueta define el inicio y el final de un documento HTML. Es el contenedor principal que indica al navegador que el contenido dentro de ella es un documento HTML. 
    
                        <html lang="es">
    HTML
  • <head>: Contiene metadatos sobre el documento, como el título, la vinculación de hojas de estilo y scripts. Estos datos no son visibles para el usuario, pero son cruciales para el SEO y el funcionamiento del documento. 
    
                        <head>
    
                          <title>Mi Página Web</title>
    
                        </head>
    Mi Página Web
  • <title>: Define el título del documento, que aparece en la pestaña del navegador y es importante para la optimización en motores de búsqueda (SEO). 
    
                        <title>Bienvenido a Mi Página</title>

    Bienvenido a Mi Página

  • <meta>: Proporciona metadatos sobre el HTML, como el conjunto de caracteres del documento, la descripción y las palabras clave. Es esencial para mejorar el SEO y la accesibilidad. 
    
                        <meta charset="UTF-8">
    
                        <meta name="description" content="Descripción de mi página web.">

    Descripción de mi página web.

  • <link>: Se utiliza para vincular a recursos externos, como hojas de estilo CSS, que son necesarias para dar formato al documento. 
    
                        <link rel="stylesheet" href="styles.css">
  • <style>: Permite definir estilos CSS internos que se aplican a los elementos del documento. Se usa principalmente para estilos que son específicos de la página actual. 
    
                        <style>
    
                          body { background-color: #f0f0f0; }
    
                        </style>
    Contenido con fondo gris claro.
  • <script>: Define o enlaza un script de JavaScript, que puede ser utilizado para agregar interactividad y funcionalidad dinámica al documento. 
    
                        <script src="script.js"></script>
  • <body>: Contiene el contenido visible del documento, como texto, imágenes y otros elementos HTML. Es la parte más importante, ya que es lo que los usuarios ven al visitar la página. 
    
                        <body>
    
                          <h1>Bienvenido a Mi Página Web</h1>
    
                        </body>

    Bienvenido a Mi Página Web

Sección y Estructura del Contenido

  • <header>: Define el encabezado de una sección o del documento, que generalmente contiene títulos, logotipos y elementos de navegación. 
    
                        <header><h1>Título Principal</h1></header>

    Título Principal

  • <nav>: Contenedor para enlaces de navegación, que ayuda a los usuarios a moverse entre diferentes secciones de un sitio web. 
    
                        <nav><a href="pagina1.html">Página 1</a><a href="pagina2.html">Página 2</a></nav>
    Página 1 Página 2
  • <section>: Define una sección genérica de un documento, que puede contener un conjunto temático de contenido. 
    
                        <section><h2>Título de la Sección</h2><p>Contenido de la sección.</p></section>

    Título de la Sección

    Contenido de la sección.

  • <article>: Representa un contenido independiente y autocontenido que puede ser distribuido o reutilizado de manera independiente (como un artículo de noticias). 
    
                        <article><h2>Título del Artículo</h2><p>Este es un artículo.</p></article>

    Título del Artículo

    Este es un artículo.

  • <aside>: Representa contenido adicional que está relacionado de manera indirecta con el contenido principal, como un bloque de información, una barra lateral o una nota. 
    
                        <aside>Este es un contenido adicional.</aside>
    Este es un contenido adicional.
  • <footer>: Define el pie de página de un documento o sección, que generalmente contiene información sobre derechos de autor, enlaces de contacto y otra información relacionada. 
    
                        <footer><p>© 2024 Mi Página Web</p></footer>

    © 2024 Mi Página Web

Encabezados y Texto

  • <h1> - <h6>: Encabezados que definen títulos y subtítulos en un documento HTML. <h1> es el más importante y <h6> el menos importante. 
    
                        <h1>Título Principal</h1>

    Título Principal

  • <p>: Define un párrafo de texto. Se utiliza para agrupar contenido textual en bloques. 
    
                        <p>Este es un párrafo de texto.</p>

    Este es un párrafo de texto.

  • <blockquote>: Se utiliza para citar en bloque. Generalmente se usa para citas largas o importantes. 
    
                        <blockquote>Esta es una cita en bloque.</blockquote>
    Esta es una cita en bloque.
  • <pre>: Define texto que se muestra con formato preformateado, manteniendo espacios y saltos de línea. 
    
                        <pre>Texto preformateado</pre>
                    
    Texto preformateado
  • <b>: Muestra texto en negrita sin darle un énfasis semántico. Es puramente visual. 
    
                        <b>Texto en negrita</b>
    Texto en negrita
  • <strong>: Muestra texto fuerte con un énfasis semántico. Se considera importante para el contenido. 
    
                        <strong>Texto fuerte</strong>
    Texto fuerte
  • <i>: Muestra texto en cursiva sin dar un énfasis semántico. Usado generalmente para texto estilizado. 
    
                        <i>Texto en cursiva</i>
    Texto en cursiva
  • <em>: Muestra texto enfatizado. Indica que el texto tiene un énfasis especial. 
    
                        <em>Texto enfatizado</em>
    Texto enfatizado
  • <mark>: Resalta texto, indicado que es relevante o importante. 
    
                        <mark>Texto resaltado</mark>
    Texto resaltado
  • <small>: Muestra texto más pequeño que el texto normal. 
    
                        <small>Texto pequeño</small>
    Texto pequeño
  • <sup>: Define texto como superíndice, que aparece elevado en relación al texto normal. 
    
                        <sup>Texto en superíndice</sup>
    Texto en superíndice
  • <sub>: Define texto como subíndice, que aparece por debajo de la línea base. 
    
                        <sub>Texto en subíndice</sub>
    Texto en subíndice
  • <br>: Inserta un salto de línea, separando el texto en líneas diferentes. 
    
                        <br>
    Texto antes del salto
    Texto después del salto
  • <hr>: Dibuja una línea horizontal en el documento, comúnmente utilizada como separador. 
    
                        <hr>
  • <abbr>: Define una abreviatura o acrónimo, que puede tener un atributo "title" para proporcionar más información. 
    
                        <abbr title="Asociación Internacional de Estudiantes">AIE</abbr>
    AIE

Listas

  • <ul>: Define una lista desordenada. Los elementos de la lista se marcan con viñetas. 
    
                        <ul>
                            <li>Elemento 1</li>
                            <li>Elemento 2</li>
                        </ul>
    • Elemento 1
    • Elemento 2
  • <ol>: Define una lista ordenada. Los elementos de la lista se numeran automáticamente. 
    
                        <ol>
                            <li>Elemento 1</li>
                            <li>Elemento 2</li>
                        </ol>
    1. Elemento 1
    2. Elemento 2
  • <dl>: Define una lista de definiciones. Contiene términos y sus descripciones. 
    
                        <dl>
                            <dt>Término 1</dt>
                            <dd>Descripción del término 1</dd>
                        </dl>
    Término 1
    Descripción del término 1

Enlaces e Imágenes

  • <a>: Define un enlace a otra página o recurso. Se utiliza el atributo href para especificar la URL del destino. 
    
                        <a href="https://www.ejemplo.com">Visitar Ejemplo</a>
    Visitar Ejemplo
  • <img>: Inserta una imagen en el documento. Se requiere el atributo src para la ruta de la imagen y alt para una descripción alternativa. 
    
                        <img src="img/imagen.jpg" alt="Descripción de la imagen">
  • <figure>: Contenedor para contenido gráfico, como imágenes o diagramas, que puede incluir una leyenda opcional. 
    
                        <figure>
    
                              <img src="img/imagen.jpg" alt="Descripción">
    
                              <figcaption>Leyenda de la imagen</figcaption>
    
                        </figure>
  • <figcaption>: Proporciona una leyenda para el contenido gráfico dentro de un elemento <figure>. 
    
                        <figcaption>Esta es una leyenda</figcaption>

Tablas

  • <table>: Define una tabla en el documento. Es el contenedor principal para todos los elementos de la tabla. 
    
                        <table>
    
                        </table>
    Encabezado 1 Encabezado 2
    Contenido 1 Contenido 2
  • <tr>: Define una fila en la tabla. Debe estar contenida dentro de una etiqueta <table>. 
    
                        <tr>
    
                              <td>Contenido de la celda</td>
    
                        </tr>
  • <th>: Define una celda de encabezado en la tabla, que generalmente se usa para títulos de columnas y filas. El texto en las celdas de encabezado se muestra en negrita y centrado por defecto. 
    
                        <th>Encabezado de la Tabla</th>
  • <td>: Define una celda de datos en la tabla, que puede contener texto, imágenes u otros elementos HTML. 
    
                        <td>Contenido de la celda</td>

Formularios

  • <form>: Contenedor para elementos de formulario. Se utiliza para agrupar y enviar datos a un servidor. 
    
                        <form action="enviar.php" method="post">
    
                        </form>
  • <input>: Campo de entrada de datos, que puede ser de varios tipos, como texto, contraseña, radio, etc. 
    
                        <input type="text" name="nombre">
  • <textarea>: Área de texto que permite ingresar múltiples líneas de texto. Es ideal para comentarios o descripciones. 
    
                        <textarea name="comentarios"></textarea>
  • <select>: Lista desplegable que permite al usuario seleccionar una opción de un conjunto de opciones. 
    
                        <select name="opcion">
    
                              <option value="1">Opción 1</option>
    
                        </select>
  • <button>: Botón que puede ser utilizado para enviar formularios o realizar otras acciones. 
    
                        <button type="submit">Enviar</button>
  • <label>: Etiqueta para un elemento de formulario, que proporciona una descripción accesible para el campo correspondiente. 
    
                        <label for="nombre">Nombre:</label>
    
                        <input type="text" id="nombre">
